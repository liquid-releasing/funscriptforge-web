# FunscriptForge — Releases

This repository hosts the official release artifacts for **FunscriptForge**, a structure-aware post-processor for funscripts.

---

## What is FunscriptForge?

FunscriptForge analyzes the motion structure of any funscript — breaking it down into phases, cycles, patterns, phrases, and BPM transitions — and gives you precise, per-phrase control over how that motion is transformed.

Whether you want to smooth out a rough draft, boost intensity in specific sections, or standardize timing across a long script, FunscriptForge lets you make targeted edits with live preview and full undo/redo, then export a clean finished file.

**Key features:**

- **Structural analysis** — automatically maps phases → cycles → patterns → phrases → BPM transitions
- **Per-phrase transforms** — apply a different processing strategy to each phrase independently
- **Live phrase preview** — HTML5 media player synced to the current phrase so you hear exactly what you changed
- **Pattern catalog** — build a named library of motion patterns across multiple funscripts
- **50-level undo/redo** — experiment freely without fear
- **One-click export** — review the full change log before committing to the output file
- **Fully local** — runs entirely on your machine; no internet connection required after install
